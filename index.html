GT7 Online Dealerships & Info

Note because I know it's a bit confusing:
"X More Days Remaining" means "this day and X more" left in the dealership.
ie. "2 More Days Remaining" -> "1 More Day Remaining" -> "Last Day Remaining".
Hope that clears up any confusion.

If you want to make a bot that uses this data, a JSON file is built from the data and can be found at:
ddm999.github.io/gt7info/data.json
This will update whenever the site is updated.
Also, if you're wondering, the site is updated manually.

All dates are (YY-MM-DD)
- Content updated for v1.23. (22-09-30)
- Content updated for v1.20. (22-08-22)
- "Estimates" are now accurate. Note that PD could still decide to diverge from them for no reason, it's just incredibly unlikely. (22-06-28)
- A lot of sections have been split up into their own pages. (22-06-27)

Breaking data.json changes:
- `daysvisible` in a `car` entry has been removed. (22-06-28)
 - A new setting for daily race `cartype` has been added: `pp`, for a PP limit. (22-06-20)
- A new setting for daily race `starttype` has been added: `grid_with_false_start`. (22-05-23)
- A new setting for daily race `cartype` has been added: `both`, for both a category and specific car restrictions. (22-05-02)
- The `estimatedays` field for dealership data has been adjusted to make more sense. (22-04-20) It's now consistently "the days remaining shown on the site, plus one".
- Booleans have all been changed to actual booleans instead of y/n strings. (22-03-24)
- For a short time, daily race specific cars were given as db IDs: that's been reverted back to names. (22-04-04)
